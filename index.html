Description

HBnB is a full-stack project that includes a web server, RESTful API, and a static HTML front-end. The project is structured to manage various tasks related to the development and deployment of a clone of the AirBnB web application.

Table of Contents

  • 1. Environment
  • 2. Installation
  • 3. Usage
  • 4. Testing

1. Environment

The project runs in a Linux environment and is tested on Ubuntu 14.04 LTS using Python 3.

2. Installation

Clone the repository:

git clone https://github.com/GuerinTifenn/holbertonschool-hbnb.git

3. Usage

To use the application, navigate to the project directory and run the main application file:

./console.py

4. Testing

Run the unit tests:

python3 -m unittest discover tests
